## ✨ Features

### 🔧 Dimensional Matter Assembler (DMA)
The heart of UFO Future — a fully custom AE2-powered machine with:
- **9-slot shapeless input grid** with fluid inputs and dual output slots
- **Dynamic thermal system** with heat zones (Safe → Hazard → Meltdown)
- **13 catalyst upgrades** across 4 families affecting speed, energy, stability, and bonus drops
- **Meltdown mechanics** with 5-second countdown, alarm sounds, red warnings, and global chat alerts broadcasting explosion coordinates
- **Passive & active cooling** via multiple coolant fluids with different efficiencies
- **JEI integration** with custom GUI and formatted energy display (K/M/G)
- **KubeJS support** for fully scripted recipe creation and modification

### ⚔️ Transformable Energy Multi-Tool
A single RF-powered tool that morphs between **10 tool types** via scroll cycling:
- Staff → Sword → Pickaxe → Axe → Shovel → Hoe → **Hammer** → Greatsword → Fishing Rod → Bow
- **Auto-Smelt** toggle on Pickaxe and Hammer
- **Area Mining** (1×1 / 3×3 / 5×5 / 7×7) on Hammer with smart energy management
- **Fast-Draw** mode on Bow
- **3×3 tilling** with auto-replant on Hoe
- Animated rainbow name effect on all tools

### 🛡️ Tiered Armor System

| Set | Tier | Key Abilities |
|-----|------|---------------|
| **Thermal Resistor Exosuit** | Mid-game | Fire/lava immunity, +15% mining speed, DMA heat protection |
| **UFO Armor** | Endgame | Resistance X, creative flight, night vision, +20 hearts (RF-powered) |

### 💾 BigInteger Storage Cells
Custom AE2 storage cells with capacities far beyond vanilla limits:

| Series | Type | Tiers |
|--------|------|-------|
| **White Dwarf** | Items | Echo (40M) → Beacon (100M) → Nexus (250M) → Core (750M) → Singularity (∞) |
| **Neutron Star** | Fluids | Same tier progression |
| **Infinity Cells** | Single-resource | Unlimited storage for 30+ resources (vanilla, AE2, Mekanism, dyes) |

All cells render as proper 3D models inside ME Drives.

### 🧪 12 Custom Fluids
Liquid Starlight · Gelid Cryotheum · Temporal Fluid · Spatial Fluid · Primordial Matter · Raw Star Matter Plasma · Transcending Matter · UU Matter · UU Amplifier · White Dwarf/Neutron Star/Pulsar Fragment Fluids

### 🌐 Internationalization

## 🚀 Getting Started

### Requirements
- **Minecraft** 1.21.1
- **NeoForge** 21.1+
- **Applied Energistics 2** (required)
- **AE2 Addon Lib** (required)
- **Mekanism** (optional — enables additional Infinity Cells)

### Installation
1. Download the latest release from [CurseForge](#) or [Releases](https://github.com/Raishxn/UFO-Future-1.21.1/releases)
2. Place the `.jar` file in your `mods/` folder
3. Ensure AE2 and AE2 Addon Lib are also installed
4. Launch the game!

### Your First Steps
1. Craft an **Obsidian Matrix** (8× Obsidian + 1× Ender Eye)
2. Build a **Graviton Plated Casing**
3. Craft the **Dimensional Matter Assembler** and connect it to your AE2 network
4. Start with **Gelid Cryotheum** as your first coolant
5. Progress through White Dwarf → Neutron Star → Pulsar materials

## 🔧 For Modpack Developers

### KubeJS Integration
UFO Future exposes its recipe type `ufo:dimensional_assembly` for full KubeJS scripting. You can:
- Add custom DMA recipes with item/fluid inputs and outputs
- Remove or replace existing recipes
- Use tags as ingredients
- See the [KubeJS documentation](https://raishxn.github.io/UFO-Future-1.21.1/en/kubejs-recipes/) for examples

### Configuration
The mod provides configuration options via `ufo-common.toml` for tuning machine behavior, energy values, and thermal thresholds.
