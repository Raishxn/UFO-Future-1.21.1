    合成进阶

    UFO Future 拥有深度互联的进阶系统。一切始于维度物质装配器 (DMA) 和黑曜石矩阵。

    1. 基础

    需要: - 运行中的 Applied Energistics 2 (AE2) 网络 - 为 AE2 网络供能的发电设备 - 基础 AE2 处理器（工程、计算、逻辑） - 下界合金、黑曜石和末影之眼

    步骤 1：黑曜石矩阵

    在标准工作台合成：8x 黑曜石环绕 1x 末影之眼。

    步骤 2：DMA

    • 引力子镀层外壳: 铁块 + 工程处理器 + 黑曜石矩阵 + 黑曜石
    • DMA 方块: 黑曜石矩阵 + 工程处理器 + 引力子外壳 + AE2 控制器 + 末影之眼

    2. 基础流体与前驱体

    步骤 3：极寒冰晶（冷却液）

    1. 暴雪粉 (DMA): 水 + 雪球 + 浮冰 → 4x 暴雪粉
    2. 冰晶粉 (DMA): 2x 暴雪粉 + 雪球 + 2x 红石 + 水 → 2x 冰晶粉
    3. 极寒冰晶 (DMA): 4x 冰晶粉 + 水 → 10,000 mB

    步骤 4：液态星光（基础流体）

    1. 在水中处理黑曜石矩阵 → 废料
    2. 9x 废料 → 废料箱
    3. 处理废料箱 → UU 放大器
    4. 4x 黑曜石矩阵 + 16x 地狱之星 + 100mB UU 放大器 → 液态星光

    3. 星体材料

    步骤 5：白矮星

    2x 下界合金锭 + 4x 蓝冰 + 4x 天空石 + 极寒冰晶 → 白矮星碎片锭

    步骤 6：中子星

    4x 白矮星锭 + 地狱之星 + 黑曜石矩阵 + 极寒冰晶 → 中子星碎片锭

    步骤 7：脉冲星

    2x 中子星锭 + 2x 磁石 + 4x 避雷针 + 极寒冰晶 → 脉冲星碎片锭

    4. 组件与处理器

    步骤 8：处理器

    1. 铁块 + 黑曜石矩阵 + 液态星光 → 维度处理器压模
    2. 白矮星锭 + 压模 + 液态星光 → 印刷维度处理器
    3. 印刷处理器 + 硅片 + 福鲁伊克斯粉 + 液态星光 → 维度处理器

    步骤 9：组件矩阵

    1. Phase Shift: Echo 单元 / T1 催化剂
    2. Hyper Dense: Beacon 单元 / T2 催化剂
    3. Tesseract: Nexus 单元 / T3 催化剂
    4. Event Horizon: Core 单元
    5. Cosmic String: Singularity 单元 / 无限单元

    5. 物质链

    步骤 10：物质球

    1. 9x 中子星锭 → 中子球
    2. 中子球 + 2x 量子异常 → 富集中子球

    步骤 11：基础物质

    1. 富集中子球 + UU 物质 → 原始物质
    2. 原始物质 + 64x 铁块 + UU 物质 → 实体物质

    步骤 12：高级物质

    实体物质 + UU 水晶 + 星体方块 = 白矮星/中子星/脉冲星物质

    步骤 13：暗物质

    每种高级物质 16 个 + 超越物质 → 暗物质

    6. 终局目标

    1. UFO 装甲: 神模式装甲，含创造飞行。
    2. UFO 法杖: 覆盖所有工具需求的多功能工具。
    3. 维度催化剂: 即时合成，无消耗。
    4. 无限单元: 无限量存储。
