    Crafting Progression

    UFO Future has a deep, interconnected progression system. Everything starts with the Dimensional Matter Assembler (DMA) and the Obsidian Matrix.

    1. The Foundation

    Before you can start with UFO Future, you need: - A functioning Applied Energistics 2 (AE2) network - Power generation to feed the AE2 network - Basic AE2 processors (Engineering, Calculation, Logic) - Netherite, Obsidian, and Ender Eyes

    Step 1: Obsidian Matrix

    Craft the Obsidian Matrix in a standard crafting table: - 8x Obsidian around 1x Ender Eye - This is the base component for almost everything in the mod.

    Step 2: The DMA

    Craft the Dimensional Matter Assembler block: - Graviton Plated Casing: Iron Blocks + Engineering Processor + Obsidian Matrix + Obsidian - DMA Block: Obsidian Matrix + Engineering Processor + Graviton Plated Casing + AE2 Controller + Ender Eye

    Place the DMA and connect it to your AE2 network with cables. It needs AE power to function.

    2. Basic Fluids & Precursors

    Before you can make advanced materials, you need coolant and base fluids.

    Step 3: Gelid Cryotheum (Coolant)

    1. Dust Blizz (DMA): Water + Snowball + Packed Ice → 4x Dust Blizz
    2. Dust Cryotheum (DMA): 2x Dust Blizz + Snowball + 2x Redstone + Water → 2x Dust Cryotheum
    3. Gelid Cryotheum (DMA): 4x Dust Cryotheum + Water → 10,000 mB Gelid Cryotheum Use this as your early-game coolant.

    Step 4: Liquid Starlight (Base Fluid)

    1. Process Obsidian Matrix in water to get Scrap.
    2. Compress 9x Scrap into a Scrap Box.
    3. Process Scrap Box to get UU Amplifier.
    4. Combine 4x Obsidian Matrix + 16x Nether Star + 100mB UU Amplifier → Liquid Starlight.

    3. Stellar Materials

    Now you can start forging stellar fragments.

    Step 5: White Dwarf

    • 2x Netherite Ingot + 4x Blue Ice + 4x Sky Stone + Gelid Cryotheum → White Dwarf Fragment Ingot
    • Make White Dwarf Blocks and Fluid.

    Step 6: Neutron Star

    • 4x White Dwarf Ingot + Nether Star + Obsidian Matrix + Gelid Cryotheum → Neutron Star Fragment Ingot

    Step 7: Pulsar

    • 2x Neutron Star Ingot + 2x Lodestone + 4x Lightning Rod + Gelid Cryotheum → Pulsar Fragment Ingot

    4. Components & Processors

    Advanced crafting requires specialized matrices.

    Step 8: Processors

    1. Iron Blocks + Obsidian Matrix + Liquid Starlight → Dimensional Processor Press
    2. White Dwarf Ingot + Press + Liquid Starlight → Printed Dimensional Processor
    3. Printed Processor + Silicon Print + Fluix Dust + Liquid Starlight → Dimensional Processor

    Step 9: Component Matrices

    1. Phase Shift: 2x Dimensional Processor + 256K Component + Glass + Nether Star = For Echo Cells / T1 Catalysts
    2. Hyper Dense: Phase Shift + White Dwarf Matter = For Beacon Cells / T2 Catalysts
    3. Tesseract: Hyper Dense + Neutron Star Matter = For Nexus Cells / T3 Catalysts
    4. Event Horizon: Tesseract + Pulsar Matter = For Core Cells
    5. Cosmic String: Event Horizon + Dark Matter = For Singularity Cells / Infinity Cells

    5. The Matter Chain

    To upgrade your components and craft endgame gear, you need to compress matter.

    Step 10: Matter Spheres

    1. 9x Neutron Star Ingot → Neutronium Sphere
    2. Neutronium Sphere + 2x Quantum Anomaly → Enriched Neutronium Sphere Note: Quantum Anomaly requires Pulsar Dust + Obsidian Matrix + Pulsar Fluid.

    Step 11: Base Matter

    1. Enriched Neutronium Sphere + UU Matter → Proto Matter
    2. Proto Matter + 64x Iron Block + UU Matter → Corporeal Matter

    Step 12: High-Tier Matter

    Combine Corporeal Matter with a UU Matter Crystal and a Stellar Block to get: - White Dwarf Matter - Neutron Star Matter - Pulsar Matter

    Step 13: Dark Matter

    Combine 16 of each high-tier matter type + Transcending Matter fluid to get Dark Matter.

    6. Endgame Goals

    Once you have Dark Matter and Cosmic String Components, you can craft:

    1. UFO Armor: God-mode armor with creative flight (from Thermal Resistor base + Enriched Neutronium Spheres).
    2. UFO Staff: The multi-tool that covers every tool need.
    3. Dimensional Catalyst: The creative-mode upgrade for the DMA that removes all energy costs and makes crafting instant.
    4. Infinity Cells: Unlimited storage for bulk resources.
