# Progresión de Fabricación

UFO Future tiene un sistema de progresión profundo e interconectado. Todo comienza con el Ensamblador de Materia Dimensional (DMA) y la Matriz de Obsidiana.

---

## 1. Los Cimientos

Necesitas:
- Una red **Applied Energistics 2** (AE2) funcionando
- Generación de energía para alimentar la red
- Procesadores básicos AE2 (Ingeniería, Cálculo, Lógica)
- **Netherite**, **Obsidiana** y **Ojos de Ender**

### Paso 1: Matriz de Obsidiana
8x Obsidiana alrededor de 1x Ojo de Ender (mesa de crafteo estándar).

### Paso 2: El DMA
- **Carcasa Gravitón**: Bloques Hierro + Procesador Ingeniería + Matriz Obsidiana + Obsidiana
- **Bloque DMA**: Matriz Obsidiana + Procesador Ingeniería + Carcasa Gravitón + Controlador AE2 + Ojo Ender

---

## 2. Fluidos y Precursores Básicos

### Paso 3: Criotheum Gélido (Refrigerante)
1. **Polvo Blizz** (DMA): Agua + Bola de Nieve + Hielo Compacto → 4x Polvo Blizz
2. **Polvo Criotheum** (DMA): 2x Polvo Blizz + Bola de Nieve + 2x Redstone + Agua → 2x Polvo Criotheum
3. **Criotheum Gélido** (DMA): 4x Polvo Criotheum + Agua → 10,000 mB

### Paso 4: Luz Estelar Líquida (Fluido Base)
1. Procesa Matriz Obsidiana en agua → **Chatarra**
2. 9x Chatarra → **Caja de Chatarra**
3. Caja de Chatarra → **Amplificador UU**
4. 4x Matriz Obsidiana + 16x Estrella Nether + 100mB Amplificador UU → **Luz Estelar Líquida**

---

## 3. Materiales Estelares

### Paso 5: Enana Blanca
2x Netherite + 4x Hielo Azul + 4x Piedra del Cielo + Criotheum → **Lingote Fragmento Enana Blanca**

### Paso 6: Estrella de Neutrones
4x Lingote EB + Estrella Nether + Matriz Obsidiana + Criotheum → **Lingote Fragmento Estrella de Neutrones**

### Paso 7: Púlsar
2x Lingote EN + 2x Magnetita + 4x Pararrayos + Criotheum → **Lingote Fragmento de Púlsar**

---

## 4. Componentes y Procesadores

### Paso 8: Procesadores
1. Bloques Hierro + Matriz Obsidiana + Luz Estelar → **Prensa Procesador Dimensional**
2. Lingote EB + Prensa + Luz Estelar → **Procesador Dimensional Impreso**
3. Procesador Impreso + Silicio + Polvo Fluix + Luz Estelar → **Procesador Dimensional**

### Paso 9: Matrices de Componentes
1. **Phase Shift**: Para Celdas Echo / Catalizadores T1
2. **Hyper Dense**: Para Celdas Beacon / Catalizadores T2
3. **Tesseract**: Para Celdas Nexus / Catalizadores T3
4. **Event Horizon**: Para Celdas Core
5. **Cosmic String**: Para Celdas Singularity / Celdas Infinitas

---

## 5. La Cadena de Materia

### Paso 10: Esferas
1. 9x Lingote EN → **Esfera de Neutronio**
2. Esfera + 2x Anomalía Cuántica → **Esfera Neutronio Enriquecida**

### Paso 11: Materia Base
1. Esfera Enriquecida + UU → **Proto Materia**
2. Proto Materia + 64x Bloque Hierro + UU → **Materia Corpórea**

### Paso 12: Materia de Alto Nivel
Corpórea + Cristal UU + Bloque Estelar = **Materia EB/EN/Púlsar**

### Paso 13: Materia Oscura
16 de cada materia de alto nivel + Materia Trascendente → **Materia Oscura**

---

## 6. Objetivos Endgame

1. **Armadura UFO**: Modo dios con vuelo creativo.
2. **Bastón UFO**: La multi-herramienta definitiva.
3. **Catalizador Dimensional**: Fabricación instantánea sin costos.
4. **Celdas Infinitas**: Almacenamiento ilimitado.
