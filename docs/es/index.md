# UFO Future Mod — Wiki

Bienvenido a la documentación oficial de **UFO Future**, un addon avanzado para AE2 en Minecraft 1.21.1 (NeoForge).

UFO Future introduce materiales estelares endgame, una poderosa máquina personalizada, herramientas energéticas transformables, conjuntos de armadura por niveles, celdas de almacenamiento BigInteger, celdas infinitas y un sistema profundo de progresión de materiales — todo integrado con Applied Energistics 2.
