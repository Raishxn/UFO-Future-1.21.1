# Mega Storage

UFO Future expande la infraestructura de crafting de AE2 con mega storages y mega co-processors.

## Mega Crafting Storages

El mod incluye bloques de storage cada vez mas grandes para trabajos extremos de autocrafting.

- Event Horizon Crafting Storage
- Quantum Drive Matrix Crafting Storage
- Tesseract Unit Crafting Storage
- Dimensional Storage Cube Crafting Storage
- Hyper-Dense Stellar Crafting Storage

## Mega Co-Processors

Los mega co-processors aumentan la cantidad de trabajo que AE2 puede planificar al mismo tiempo.

- Singularity Accelerator
- Qubit Co-Processor
- Tesseract Logic Unit
- Subspace Co-Processor
- Galactic Core Co-Processor

## Papel En La Progresion

Estos bloques no sustituyen al DMA ni a los multibloques.

- **DMA** y **QMF** producen muchas de sus piezas avanzadas
- Los **multibloques cuanticos** ejecutan recetas en escala de fabrica
- El hardware de mega storage evita que planes enormes ahoguen el crafting de AE2
