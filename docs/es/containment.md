# Containment

UFO Future usa dos objetos de contencion para manejar materia inestable de late game.

## Safe Containment Matter

La **Safe Containment Matter (SCM)** es la base estable para flujos con materia peligrosa.

- Se usa como componente seguro para craftings de alto riesgo
- Va antes de la etapa de capsula
- Aparece en recetas que necesitan estabilizar materia volatil

## Aether Containment Capsule

La **Aether Containment Capsule (ACC)** eleva la SCM a una carcasa portatil de contencion.

- Sirve para mover o craftear materiales peligrosos
- Evita tratar items fragiles de progresion como componentes comunes
- Es el paso superior de contencion en la linea de materia
