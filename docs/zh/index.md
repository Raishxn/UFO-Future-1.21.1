# UFO Future 模组 — 维基

欢迎来到 **UFO Future** 的官方文档，这是一个用于 Minecraft 1.21.1 (NeoForge) 的高级 AE2 附属模组。

UFO Future 引入了终局星体材料、强大的自定义机器、可变形能量工具、分级装甲套装、BigInteger 存储单元、无限单元以及深度材料进阶系统 — 全部与 Applied Energistics 2 集成。
