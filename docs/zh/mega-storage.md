# Mega Storage

UFO Future extends AE2 crafting infrastructure with very large crafting storages and mega co-processors.

## Mega Crafting Storages

The mod includes progressively larger storage blocks for extreme autocrafting jobs.

- Event Horizon Crafting Storage
- Quantum Drive Matrix Crafting Storage
- Tesseract Unit Crafting Storage
- Dimensional Storage Cube Crafting Storage
- Hyper-Dense Stellar Crafting Storage

## Mega Co-Processors

Mega co-processors increase how much work AE2 can schedule at the same time.

- Singularity Accelerator
- Qubit Co-Processor
- Tesseract Logic Unit
- Subspace Co-Processor
- Galactic Core Co-Processor

## Role In Progression

These blocks do not replace the DMA or the multiblocks.

- **DMA** and **QMF** produce many of their advanced parts
- **Quantum multiblocks** execute recipes at factory scale
- Mega storage hardware keeps giant AE2 jobs from stalling
