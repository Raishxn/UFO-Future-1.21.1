# 维度物质装配器 (DMA)

**维度物质装配器** 是 UFO Future 的核心机器。模组中所有高级物品、流体、材料和组件都通过 DMA 合成。它是一台由 AE2 供能的机器，支持无序物品输入、流体输入、冷却系统和催化剂升级。

---

## 合成 DMA

```
O P O
C M C
O E O
```

| 符号 | 物品 |
|------|------|
| O | 黑曜石矩阵 |
| P | AE2 工程处理器 |
| C | 引力子镀层外壳 |
| M | AE2 控制器方块 |
| E | 末影之眼 |

> **前置条件**：需要先获取 AE2 和引力子镀层外壳。

---

## GUI 概览

| 区域 | 描述 |
|------|------|
| **9 格输入网格** | 无序物品输入（顺序无关） |
| **2 个输出槽** | 物品/流体输出在此显示 |
| **冷却液箱** (槽位 2) | 玩家管理的冷却流体 — 配方不消耗 |
| **基础流体箱** (槽位 3) | 配方所需流体输入 — 配方会消耗 |
| **4 个升级槽** | 用于催化剂升级卡 |
| **进度条** | 显示加工进度 |
| **能量显示** | 显示当前 AE 能量储量和消耗速率 |
| **温度计** | 显示当前热量 vs 最大容量 |

---

## 工作原理

1. 将物品放入 9 格输入网格。**顺序无关** — 所有配方都是无序的。
2. 如配方需要，向右侧流体箱（槽位 3）添加**基础流体**。
3. 向左侧流体箱（槽位 2）添加**冷却液**以防止过热。
4. 在 4 个升级槽中插入**催化剂**以修改性能。
5. **连接 AE2 能源** — DMA 是 AE2 网络方块实体。
6. 当找到有效配方匹配且能量充足时开始加工。

## 温控系统

DMA 在加工过程中产生热量,必须冷却以防止灾难性故障。

### 热量区域

| 区域 | 阈值 | 效果 |
|------|------|------|
| **安全** | < 50% 容量 | 正常运行 |
| **危险** | ≥ 50% 容量 | 旋转火焰粒子环，烧伤无热防护装甲的附近玩家 |
| **熔毁** | 100% 容量 | 5 秒倒计时，然后**爆炸**摧毁机器和周围方块 |

### 热量生成
- **工作中**: 每 2 tick +1 HU（≈ +10 HU/s），乘以催化剂热量修正
- **空闲中**: 被动冷却 每 40 tick −1 HU（≈ −0.5 HU/s）

### 冷却液效率

| 冷却液 | 效率 | 备注 |
|--------|------|------|
| **时间流体** | 100 HU / 1 mB | 最高效 — 终局冷却液 |
| **液态星光** | 30 HU / 1 mB | 中期良好选择 |
| **默认流体** | 15 HU / 1 mB | 任何未识别流体 |
| **极寒冰晶** | 1 HU / 200 mB | 基于体积 — 需要大量泵送 |

### 熔毁爆炸
温度达到最大容量 100% 时：
1. **5 秒倒计时**开始 (100 tick)
2. 每秒播放**警报声**
3. 向附近玩家显示**红色动作栏警告**
4. 机器**爆炸**（威力 10.0，破坏方块）
5. **全服聊天警报**广播爆炸坐标

> **热阻外骨骼装甲**提供对 DMA 热量伤害的完全免疫。参见 [装甲](armor.md)。

---

## 能量消耗

- 基础内部缓存：**500,000 AE**
- **Matterflow 催化剂** 可将缓存增加 10x/100x/1000x
- 每 tick 能耗 = `能量 / (配方时间 / 速度因子)`

---

## 自动导出

DMA 支持**自动导出**。启用后，完成的输出自动推送到相邻容器/ME 存储。
