# 工具与武器

UFO Future 引入了**可变形能量多功能工具系统**。所有工具共享一个 RF 供能的物品，可通过滚轮循环变形为不同工具类型。

---

## 多功能工具系统

- **RF 供能**：每次使用消耗 RF 能量而非耐久度
- **彩虹名称**：工具名称显示动画彩虹色效果
- **可变形**：按住工具并滚动 (Shift+滚轮) 循环切换所有工具类型
- **能量持久**：变形时保留能量
- **能量条**：物品上显示动态能量条
