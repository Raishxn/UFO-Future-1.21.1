# 装甲套装

UFO Future 添加了两套具有独特机制的装甲：**热阻外骨骼**（中期，热防护）和 **UFO 装甲**（终局，完整神模式含飞行）。

---

## 热阻外骨骼

专为保护玩家免受 DMA 危险区域热量伤害而设计。

### 属性与能力（每件）

| 特性 | 数值 |
|------|------|
| 基础护甲 | UFO 标准等级 |
| 火焰免疫 | ✅ 即时熄灭火焰 |
| +15% 挖掘效率 | 每件（属性修饰器）|
| 防火物品 | 不会被火/岩浆摧毁 |

### 全套加成（4 件）

| 加成 | 描述 |
|------|------|
| 抗性 II | 永久伤害减免 |
| 火焰/岩浆免疫 | 完全免疫 |
| DMA 热量免疫 | 安全免受 DMA 危险区伤害 |
| 热稳定性 | 完全热防护 |

> 热阻外骨骼是**唯一**能防护 DMA 热量伤害的装甲（≥50% 热量）。

### 合成

#### 热阻镀层（材料）
**DMA**: 下界合金锭 ×1 + 黑曜石矩阵 ×1 + 500 mB 极寒冰晶 · 50,000 AE, 100 tick

#### 装甲部件（DMA 配方）

| 部件 | 镀层数 | 能量 | 时间 |
|------|-------|------|------|
| 面罩（头盔）| 5 | 1,000,000 AE | 500 tick |
| 胸甲 | 8 | 1,500,000 AE | 500 tick |
| 护腿 | 7 | 1,200,000 AE | 500 tick |
| 靴子 | 4 | 800,000 AE | 500 tick |

所有部件需要 1,000 mB 极寒冰晶。

---

## UFO 装甲（终局）

终极装甲套装。需要热阻外骨骼作为基础，全套穿戴时获得神级能力。

### 全套加成（需要 RF）

| 加成 | 描述 |
|------|------|
| **抗性 X** | 近乎无敌（放大器 9）|
| **夜视** | 永久夜视 |
| **创造飞行** | 完整创造模式飞行 |
| **+40 最大生命** | 额外 20 颗心 |

### 能量消耗
- **400 RF/秒** (20 RF/tick) 每件
- 任何部件能量耗尽时所有效果立即移除

### 合成（DMA 配方）

每件：对应热阻部件 + 2× 富集中子球 + 量子异常 + 2,000 mB 超越物质
**50,000,000 AE** · 2,000 tick
