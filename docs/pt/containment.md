# Containment

UFO Future usa dois itens de contencao para lidar com materia instavel no late game.

## Safe Containment Matter

A **Safe Containment Matter (SCM)** e a base estavel para fluxos com materia perigosa.

- Serve como componente seguro para craftings de alto risco
- Vem antes da etapa de capsula
- Aparece em receitas que precisam estabilizar materia volatil

## Aether Containment Capsule

A **Aether Containment Capsule (ACC)** eleva a SCM para uma casca portatil de contencao.

- Usada para mover ou craftar materiais perigosos
- Evita tratar itens frageis de progressao como componentes comuns
- E a etapa de contencao superior da linha de materia
