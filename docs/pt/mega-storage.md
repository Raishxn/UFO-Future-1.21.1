# Mega Storage

UFO Future expande a infraestrutura de crafting da AE2 com mega storages e mega co-processors.

## Mega Crafting Storages

O mod inclui blocos de storage progressivamente maiores para jobs extremos de autocrafting.

- Event Horizon Crafting Storage
- Quantum Drive Matrix Crafting Storage
- Tesseract Unit Crafting Storage
- Dimensional Storage Cube Crafting Storage
- Hyper-Dense Stellar Crafting Storage

## Mega Co-Processors

Os mega co-processors ampliam a quantidade de trabalho que a AE2 consegue agendar ao mesmo tempo.

- Singularity Accelerator
- Qubit Co-Processor
- Tesseract Logic Unit
- Subspace Co-Processor
- Galactic Core Co-Processor

## Papel Na Progressao

Esses blocos nao substituem o DMA nem os multiblocos.

- **DMA** e **QMF** produzem muitas pecas avancadas deles
- Os **multiblocos quanticos** executam receitas em escala de fabrica
- O hardware de mega storage impede que planos gigantes engasguem o crafting da AE2
