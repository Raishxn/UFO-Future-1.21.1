# Progressao

O UFO Future agora tem uma escada de maquinas mais clara em vez de centralizar tudo no DMA para sempre.

## 1. Etapa do DMA

Comece com o **DMA** para:

- primeiros materiais avancados
- producao de coolants
- catalisadores
- primeiros componentes e processadores

Essa e a fase flexivel de bloco unico.

## 2. Etapa dos Multiblocos Universais

Quando as recipes ficarem grandes, frequentes ou caras demais para uso manual do DMA, avance para os multiblocos:

- **Quantum Matter Fabricator** para jobs grandes estilo DMA
- **Quantum Slicer** para partes impressas
- **Quantum Processor Assembler** para montagem final de processadores

Aqui o estilo de jogo muda de “alimentar uma maquina” para “deixar o AE2 despachar trabalho numa linha paralela”.

## 3. Etapa de Upgrade de Tier

Subir o tier do multibloco faz duas coisas:

- libera recipes de tier maior
- acelera automaticamente recipes de tier menor

Isso significa que as linhas antigas continuam relevantes, so que viram infraestrutura barata e rapida.

## 4. Etapa do Stellar Nexus

O **Stellar Nexus** fica acima dos multiblocos universais. Ele serve para simulacoes enormes, custos absurdos de AE e geracao em escala extrema.

Ele nao e apenas um crafter mais forte. E o capstone industrial do mod.
