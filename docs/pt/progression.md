# Progressão de Fabricação

UFO Future tem um sistema de progressão profundo e interconectado. Tudo começa com o Montador de Matéria Dimensional (DMA) e a Matriz de Obsidiana.

---

## 1. A Fundação

Antes de começar com UFO Future, você precisa:
- Uma rede **Applied Energistics 2** (AE2) funcionando
- Geração de energia para alimentar a rede AE2
- Processadores básicos AE2 (Engenharia, Cálculo, Lógica)
- **Netherite**, **Obsidiana** e **Olhos de Ender**

### Passo 1: Matriz de Obsidiana
Fabrique a **Matriz de Obsidiana** em uma mesa de crafting padrão:
- 8x Obsidiana ao redor de 1x Olho de Ender
- *Este é o componente base para quase tudo no mod.*

### Passo 2: O DMA
Fabrique o bloco do **Montador de Matéria Dimensional**:
- **Invólucro Banhado a Gráviton**: Blocos de Ferro + Processador de Engenharia + Matriz de Obsidiana + Obsidiana
- **Bloco DMA**: Matriz de Obsidiana + Processador de Engenharia + Invólucro Banhado a Gráviton + Controlador AE2 + Olho de Ender

*Coloque o DMA e conecte-o à sua rede AE2 com cabos. Ele precisa de energia AE para funcionar.*

---

## 2. Fluidos & Precursores Básicos

Antes de produzir materiais avançados, você precisa de refrigerante e fluidos base.

### Passo 3: Criotheum Gélido (Refrigerante)
1. **Pó Blizz** (DMA): Água + Bola de Neve + Gelo Compacto → 4x Pó Blizz
2. **Pó Criotheum** (DMA): 2x Pó Blizz + Bola de Neve + 2x Redstone + Água → 2x Pó Criotheum
3. **Criotheum Gélido** (DMA): 4x Pó Criotheum + Água → 10.000 mB Criotheum Gélido
   *Use como seu refrigerante no início do jogo.*

### Passo 4: Luz Estelar Líquida (Fluido Base)
1. Processe Matriz de Obsidiana em água para obter **Sucata**.
2. Comprima 9x Sucata em uma **Caixa de Sucata**.
3. Processe Caixa de Sucata para obter **Amplificador UU**.
4. Combine 4x Matriz de Obsidiana + 16x Estrela do Nether + 100mB Amplificador UU → **Luz Estelar Líquida**.

---

## 3. Materiais Estelares

Agora você pode começar a forjar fragmentos estelares.

### Passo 5: Anã Branca
- 2x Lingote de Netherite + 4x Gelo Azul + 4x Pedra do Céu + Criotheum Gélido → **Lingote de Fragmento de Anã Branca**
- Faça Blocos e Fluido de Anã Branca.

### Passo 6: Estrela de Nêutrons
- 4x Lingote de Anã Branca + Estrela do Nether + Matriz de Obsidiana + Criotheum Gélido → **Lingote de Fragmento de Estrela de Nêutrons**

### Passo 7: Pulsar
- 2x Lingote de Estrela de Nêutrons + 2x Magnetita + 4x Para-raio + Criotheum Gélido → **Lingote de Fragmento de Pulsar**

---

## 4. Componentes & Processadores

Fabricação avançada requer matrizes especializadas.

### Passo 8: Processadores
1. Blocos de Ferro + Matriz de Obsidiana + Luz Estelar Líquida → **Prensa do Processador Dimensional**
2. Lingote de Anã Branca + Prensa + Luz Estelar → **Processador Dimensional Impresso**
3. Processador Impresso + Impressão de Silício + Pó de Fluix + Luz Estelar → **Processador Dimensional**

### Passo 9: Matrizes de Componentes
1. **Phase Shift**: 2x Processador Dimensional + Componente 256K + Vidro + Estrela do Nether = Para Células Echo / Catalisadores T1
2. **Hyper Dense**: Phase Shift + Matéria de Anã Branca = Para Células Beacon / Catalisadores T2
3. **Tesseract**: Hyper Dense + Matéria de Estrela de Nêutrons = Para Células Nexus / Catalisadores T3
4. **Event Horizon**: Tesseract + Matéria de Pulsar = Para Células Core
5. **Cosmic String**: Event Horizon + Matéria Escura = Para Células Singularity / Células Infinitas

---

## 5. A Cadeia de Matéria

Para melhorar seus componentes e fabricar equipamentos endgame, você precisa comprimir matéria.

### Passo 10: Esferas de Matéria
1. 9x Lingote de Estrela de Nêutrons → **Esfera de Neutrônio**
2. Esfera de Neutrônio + 2x Anomalia Quântica → **Esfera de Neutrônio Enriquecida**
   *Nota: Anomalia Quântica requer Pó de Pulsar + Matriz de Obsidiana + Fluido de Pulsar.*

### Passo 11: Matéria Base
1. Esfera de Neutrônio Enriquecida + Matéria UU → **Protomatéria**
2. Protomatéria + 64x Bloco de Ferro + Matéria UU → **Matéria Corpórea**

### Passo 12: Matéria de Alto Tier
Combine Matéria Corpórea com um Cristal de Matéria UU e um Bloco Estelar para obter:
- **Matéria de Anã Branca**
- **Matéria de Estrela de Nêutrons**
- **Matéria de Pulsar**

### Passo 13: Matéria Escura
Combine 16 de cada tipo de matéria de alto tier + fluido de Matéria Transcendente para obter **Matéria Escura**.

---

## 6. Objetivos Endgame

Uma vez que você tenha Matéria Escura e Componentes Cosmic String, você pode fabricar:

1. **Armadura UFO**: Armadura modo deus com voo criativo (a partir da base Resistor Térmico + Esferas de Neutrônio Enriquecidas).
2. **Cajado UFO**: A multi-ferramenta que cobre toda necessidade de ferramenta.
3. **Catalisador Dimensional**: O upgrade de modo criativo para o DMA que remove todos os custos de energia e torna o crafting instantâneo.
4. **Células Infinitas**: Armazenamento ilimitado para recursos em massa.
