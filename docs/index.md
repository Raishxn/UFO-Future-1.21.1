# UFO Future Mod Wiki

Welcome to the official documentation for the **UFO Future** mod for Minecraft! 
This wiki contains instructions on how to progress through the mod, use machines, navigate AE2 integrations, and handle custom fluids and elements.

## Getting Started
UFO Future introduces extremely powerful items and fluids. Most of these involve advanced AE2 networking.
- Learn about the [Dimensional Matter Assembler](dma.md)

Explore the side menu for specific block and item guides!
