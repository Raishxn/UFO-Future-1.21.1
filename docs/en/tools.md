# Tools & Weapons

UFO Future includes a transformable energy tool line plus utility items for multiblock debugging.

## Energy Tool Line

The core UFO tools share these properties:

- Energy-powered instead of durability-based
- Transformable between multiple tool roles
- Persistent stored energy while changing form
- Progression perks such as area mining, auto-smelt or combat scaling depending on the tool

## Common Forms

- **Pickaxe**: mining progression and ore-focused utility
- **Hammer**: area mining
- **Sword / Greatsword**: combat-focused forms
- **Axe, Shovel, Hoe, Fishing Rod, Bow, Staff**: specialized utility forms

## Structure Scanner

The **Structure Scanner** is the main utility item for the multiblock line.

- Right-click a supported controller to validate the structure
- Highlights missing or invalid blocks on the client
- Prints exact block positions and expected parts in chat
- In Creative, **Shift + Right Click** can auto-build the structure shell from the default preview states
