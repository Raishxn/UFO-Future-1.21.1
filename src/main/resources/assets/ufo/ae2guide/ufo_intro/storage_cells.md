# High-Capacity Storage Cells

Welcome to the **UFO Future** cellular storage technology! 
Our storage cells transcend vanilla Applied Energistics 2 capacities, breaking the standard bit limits to hold astronomical amounts of matter.

## Multi-Tier Cosmic Cells

These cells are forged using intense astronomical matter, allowing extreme byte capacities!
Each cell type stores both extremely large item/fluid capacities with different colored LED cores indicating their tier.

- **100M Cells** *(Beacon Core)*: Foundational Hyper-Dense networks
- **250M Cells** *(Nexus Core)*: Advanced Tesseract mechanics
- **750M Cells** *(Core)*: Massive Event Horizon containment
- **Singularity Cells** *(Green Core)*: Boundless 65k types storage using Cosmic Strings.

**Infinity Cells** are endgame artifacts representing the absolute pinnacle of matter manipulation. Once crafted, they insert virtually **infinite amounts of a single resource** directly into your network without consuming actual storage.

### Fabrication Process
Infinity Cells can **only be crafted within the Dimensional Matter Assembler (DMA)**!
The intense pressure needed to forge infinite matter requires extreme conditions:
1. Obtain a **Cosmic String Component** and a **Quantum Anomaly**.
2. Supply large amounts of the raw material (e.g., 64x Water Buckets, 64x Cobblestone).
3. Inject **Transcending Matter Fluid**.
4. The DMA will synthesize the singularity into a stable permanent node.
