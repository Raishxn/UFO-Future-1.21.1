# Stellar Field Generators

<BlockImage id="ufo:stellar_field_generator_t1" scale="3"></BlockImage>

Field generators are the stabilization blocks used throughout the advanced UFO multiblock line.

- **T1** is used in early quantum structures and the Cryoforge.
- **T2** supports heavier processor assemblies.
- **T3** is reserved for the highest-end structures.
