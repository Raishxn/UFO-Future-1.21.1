# UFO Tools

UFO Future incluye herramientas energeticas transformables y utilidades para multibloques.

## Tool Line

- Herramientas con energia en lugar de durabilidad
- Varias formas: Pickaxe, Hammer, Sword, Greatsword, Axe, Shovel, Hoe, Bow, Staff y mas
- Conservan energia durante la transformacion

## Structure Scanner

- Valida estructuras multibloque
- Resalta bloques faltantes o invalidos
- Muestra posiciones y piezas esperadas
- En creativo puede auto-construir la carcasa con **Shift + clic derecho**
