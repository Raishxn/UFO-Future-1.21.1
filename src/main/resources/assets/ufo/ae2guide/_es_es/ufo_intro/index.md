# Bienvenido a UFO Future

**UFO Future** expande el endgame de AE2 con maquinas termicas, materia estelar, mega storage y automatizacion multibloque.

![Logo](../pic/logo.png)

## Lineas Principales

- **DMA**: la maquina avanzada flexible del inicio
- **Quantum Multiblocks**: automatizacion masiva con patterns y paralelismo
- **Stellar Nexus**: plataforma final de simulacion a escala extrema

## Nueva Linea Multibloque

- **Quantum Matter Fabricator**: evolucion del DMA para automatizacion AE2
- **Quantum Slicer**: preparacion masiva de partes impresas
- **Quantum Processor Assembler**: acabado masivo de procesadores
- **Quantum Pattern Hatch**: proveedor AE2 de **72 patterns**
